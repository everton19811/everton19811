# Definition of Ready (DoR)
- [ ] História com valor claro
- [ ] Critérios de aceite escritos
- [ ] Estimada pelo time
- [ ] Dependências mapeadas
- [ ] Cabe em uma sprint

# Definition of Done (DoD)
- [ ] Código revisado
- [ ] Testes automatizados
- [ ] Documentação atualizada
- [ ] Aprovado pelo PO
- [ ] Deploy em homologação
