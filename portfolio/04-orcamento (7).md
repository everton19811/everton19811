# 📊 Case — Dashboard de EVM para Portfólio de Projetos

> Implantação de um **dashboard corporativo de Earned Value Management (EVM)** para o portfólio de projetos do PMO, com Status Report semanal automatizado, semáforo RAG e projeções (EAC/ETC/VAC).

- **Cliente (anonimizado):** grande varejo B2B — PMO Corporativo
- **Duração:** 4 meses (16 sprints de 1 semana)
- **Time:** 1 PM (eu), 1 Analista de Dados, 1 Eng. de BI, 1 Sponsor (Diretor de PMO)
- **Stack:** Azure DevOps · SQL Server · Power BI · SharePoint · Power Automate

---

## 📂 Estrutura

```text
case-dashboard-evm-portfolio/
├── 01-iniciacao/
├── 02-planejamento/
├── 03-execucao/
├── 04-monitoramento-controle/
├── 05-encerramento/
└── diagrams/
```

## 🎯 Resultados

- **-60%** no tempo de consolidação do Status Report semanal (de 8h → 3h)
- **100%** dos 24 projetos do portfólio com CPI/SPI calculados semanalmente
- **+35%** de aderência às baselines após implantação (governança semanal)
- **1 fonte única** da verdade para o comitê executivo
